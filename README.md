# 📚 Mini Kitaplık

**Mini Kitaplık**, React ekosistemini ve temel konseptlerini (component yapısı, state yönetimi) öğrenmek ve pratik yapmak amacıyla geliştirilmiş tek sayfalık bir web uygulamasıdır (SPA). React ve Vite kullanılarak hızlı bir geliştirme ortamı ile inşa edilmiştir.

Kullanıcılar bu uygulama üzerinden kendi dijital kütüphanelerini oluşturabilir, yeni kitaplar ekleyebilir ve beğendikleri kitapları favorilerine alarak kolayca yönetebilirler.

🚀 **Canlı Uygulama (Vercel):** [Mini Kitaplık Canlı Uygulama](https://mini-kitaplik-seven.vercel.app/)

### ✨ Temel Özellikler
*   **Kitap Ekleme:** Sisteme yeni kitap isimleri ve detayları ekleyebilme.
*   **Favorilere Alma:** Listelenen kitaplar arasından istenilenleri favori olarak işaretleme ve filtreleme.
*   **Hızlı ve Modern Altyapı:** Vite'in sunduğu hızlı yenileme (HMR) sayesinde akıcı kullanıcı deneyimi.
*   **Öğrenme Odaklı Geliştirme:** React component mimarisi ve state (durum) yönetiminin temel düzeyde uygulanması.

### 🛠️ Kullanılan Teknolojiler
*   **Frontend:** React (Hooks)
*   **Build Aracı:** Vite
*   **Dağıtım (Deployment):** Vercel
